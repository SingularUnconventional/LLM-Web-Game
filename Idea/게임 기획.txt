🧭 게임 개요



장르: 감정 중심 인터랙티브 심리 채팅 게임

플랫폼: 웹 기반 (모바일 우선 대응), PC 가능

진행 방식: 심리 테스트 → 캐릭터 꿈 접속 → 채팅을 통한 고민 해결 → 감정 조각 수집 → 페르소나 완성

플레이 시간: 1 캐릭터당 약 3~6일 소요 (1일 1세션 기준)

UI 중심 요소: 채팅창 + 감정 로그 + 캐릭터 프로필 카드 + 꿈 속 시각적 테마 배경

📒 1. 시작 & 심리 테스트

1.1 시작 화면



"지금, 당신의 마음과 연결할 준비가 되셨나요?"

조용한 음악, 빛이 일렁이는 느낌의 인터페이스

‘시작하기’ 버튼 클릭 → 3~5문항의 심리 체크 (감정 단어 선택 + 시각적 선택지 등)

1.2 매칭 캐릭터 선정



사용자의 선택 패턴에 따라 현재 마음과 유사한 캐릭터 자동 배정

선택 기준 예시:

‘무력감’ → 백조소녀

‘지나친 책임감’ → 종이사자 소년

‘소외감’ → 유리창 마녀

1.3 시스템 안내 (비유적으로)



플레이어는 ‘꿈을 관통하는 관찰자’로서, 자신과 닮은 존재의 꿈에 들어간다

“그들은 널 모른다. 다만 네 목소리가 익숙할 뿐이야.”

🌙 2. 캐릭터 꿈 접속 & 채팅 구조

2.1 첫 접속



공간: 캐릭터의 내면을 반영한 동화적 꿈 배경 이미지 + 간단한 UI 테마

캐릭터가 등장하여 플레이어를 '꿈 속에 떠오른 존재'로 인식

첫 채팅에서 바로 고민을 털어놓는다 (매우 명확한 방식으로)

2.2 채팅 인터페이스



캐릭터 메시지: 자동 진행, 그러나 멈춤점마다 플레이어 응답 유도

플레이어 응답: 선택지 기반 (3~4개) or 텍스트 입력 기반 (후반 해금)

메시지는 실제 메신저처럼 자연스럽게 흐름 (타이핑 효과, 반응 딜레이 등)

2.3 감정 반응 시스템



플레이어의 반응에 따라 캐릭터의 감정 색이 바뀜 (노출은 간접적으로)

예: [불안 → 차분], [의심 → 신뢰], [슬픔 → 담담]

캐릭터 말투나 문장 길이, 이모티콘 사용 여부 등이 바뀌는 식으로 표현

🔁 3. 낮/밤 루프: 고민 해결 전개

3.1 밤 – 꿈 속 채팅



하루 동안 있었던 감정과 상황을 정리해 공유함

플레이어는 그 경험을 함께 되짚으며 감정을 정리해주는 역할

감정이 겹치는 순간 ‘조각’ 생성이 예고됨 (시각/음향 연출)

3.2 낮 – 현실 행동 (자동 진행)



플레이어는 볼 수 없지만, 캐릭터는 현실에서 시도한다

다음 접속 시 ‘행동 요약’ 보고서가 간결하게 제공됨

예: "오늘은 말을 꺼내봤어. 다들 대답은 했지만, 이상하게 멀었어."

3.3 반복 구조



일반적으로 3~6일 동안 캐릭터의 문제를 점진적으로 다룸

중간 전환점에서는 플레이어의 선택이 감정 진척도에 영향을 줌

플레이어의 개입 방식(공감, 직언, 침묵 유지 등)이 캐릭터에 따라 다르게 작용

🏁 4. 마무리 & 조각 수확

4.1 고민 해결 구조



문제는 단순 해결이 아닌 '관점 변화' 혹은 '감정 수용'으로 종결됨

예: "그 애들은 나랑 다르게 살아. 그래도 괜찮아."

클리어 후 캐릭터는 “이제 이 꿈엔 안 나타날 거야.”라고 작별

4.2 감정 조각



각 캐릭터로부터 특정 감정 조각 획득

조각 예시: [소속의 상처], [존재 확인], [목소리를 가진다]

조각은 플레이어의 페르소나 조각화 요소로 저장됨

💠 5. 페르소나 완성 구조 (엔딩)



모든 조각을 모으면 마지막 꿈에 접속 가능

그곳엔 플레이어 자신의 ‘상징적 자아’가 기다림

대화는 전반적으로 자문자답 구조

“넌 왜 이 꿈들을 따라왔지?”

“계속 연결되고 싶은 이유는 뭐야?”

마지막 선택: “계속 남는다 / 이제 떠난다”

이후 작은 에필로그 제공 + 조각으로 플레이어의 심상형 전시 가능

📚 보조 시스템



감정 로그 시스템: 캐릭터의 말과 감정 변화를 저장한 다이어리

조각도감: 수집한 감정 조각을 확인 가능 (비유적 표현과 해설 포함)

캐릭터 카드: 첫인상, 말투, 고민 키워드, 공간 정보 확인 가능
